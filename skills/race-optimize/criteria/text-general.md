# Text Optimization Criteria

## Hard Constraints
- **Factual accuracy**: Do not introduce factual errors contradicting the original
- **Intent preservation**: Optimized text must maintain the original purpose and core arguments

## Scoring Dimensions (10 points each)

### 1. Clarity (weight 30%)

**Anchored rubric:**
- 1-3: Less clear than original, confusing structure or ambiguous statements
- 4-5: Similar clarity to original, minor improvements
- 6-7: Noticeably clearer, better organized, fewer ambiguities
- 8-9: Excellent clarity, logical flow is immediately apparent
- 10: Crystal clear, every sentence serves a purpose, zero ambiguity

**Evaluation points:**
- Logical structure clarity
- Natural paragraph transitions
- Concise, unambiguous sentences
- Core points prominently highlighted

### 2. Professionalism (weight 25%)

**Anchored rubric:**
- 1-3: Less professional than original, casual or sloppy tone
- 4-5: Comparable professionalism, no significant improvement
- 6-7: More polished, better terminology usage
- 8-9: Highly professional, precise terminology, rigorous argumentation
- 10: Publication-ready, exemplary domain expertise

**Evaluation points:**
- Accurate terminology usage
- Rigorous and complete argumentation
- Sufficient references/evidence
- Conformity to domain conventions

### 3. Completeness (weight 25%)

**Anchored rubric:**
- 1-3: Missing key information present in original
- 4-5: Same information coverage as original
- 6-7: Better organized, some additions that strengthen the argument
- 8-9: Comprehensive, all necessary points covered with appropriate detail
- 10: Exhaustive yet concise, nothing missing, nothing superfluous

**Evaluation points:**
- All necessary points covered
- No critical information omitted
- Appropriate level of detail
- Coherent beginning-to-end structure

### 4. Readability (weight 20%)

**Anchored rubric:**
- 1-3: Harder to read than original
- 4-5: Similar reading experience
- 6-7: Smoother reading, better flow
- 8-9: Engaging and easy to follow, well-formatted
- 10: Delightful to read, perfectly adapted to target audience

**Evaluation points:**
- Target audience fit
- Language style consistency
- Reasonable formatting
- Smooth reading experience

## Sub-type Specializations

### Patent documents
- Maximize claim protection scope
- Faithful and accurate technical description
- Stable and rigorous legal language
- Well-layered dependent claims

### Technical documentation
- Steps are actionable (reader can follow along)
- Code examples are accurate and runnable
- Architecture/flow diagrams aid understanding
- Complete version/environment information

### Academic papers
- Clear innovation statement
- Rigorous reproducible methodology
- Sound experimental design
- Comprehensive related work review

### Marketing copy
- Clear value proposition
- Precise target audience
- Clear call-to-action
- Emotional resonance

## Scoring Rules
- Every score MUST include specific evidence
- Minimum 2-point spread across versions on each dimension
- Use the full scoring range

## Essence Extraction Guide
1. **Best structure** — which version's overall organization is most logical?
2. **Best paragraph** — which paragraph in each version is best written?
3. **Best expression** — which version's key argument is most precise?
4. **Best creativity** — which version has a unique expression style or argument angle?
